## Tentang Kami

**HIMASI UNMA Banten** adalah Himpunan Mahasiswa Sistem Informasi Universitas Mathla'ul Anwar Banten yang menjadi wadah pengembangan minat, bakat, kreativitas, dan kolaborasi mahasiswa Sistem Informasi.

Melalui GitHub ini, kami mengelola berbagai repository yang berkaitan dengan pengembangan teknologi, dokumentasi organisasi, sistem informasi, serta project kolaboratif mahasiswa.
